Yes. The interpretation has to be done as follows:

tam = Him
Ekadantam = the one with single "tooth/tusk"
bhaktaanaam anEkadam = that bestows countless (boons) to devotees
upaasmahE = (I am) propitiating/propitiate.

dantam = tooth/tusk.
tunDam = trunk.
(vakra = curved.)

-Sarma
