Sankara's दुस्तर्कात्सुविरम्यताम् श्रुतिमतस्तर्कोऽनुसन्धीयताम् means, if
there is logic which goes against
Sruti discard it and accept and patronise only that logic which supports
Sruti. All logic is not accepted.
