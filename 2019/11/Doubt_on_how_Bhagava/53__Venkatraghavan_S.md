Namaste,
Not sure if this is relevant, but there is a reference to shankarAchArya as
draviDa by sureshvarAchArya in the naiShkarmya siddhi 4.44.
एवं गौडैर्द्राविडैर्नः पूज्यैरयमर्थः प्रभाषितः ।
अज्ञानमात्रोपाधिः सन्नहमादिदृगीश्वर:॥

Regards
Venkatraghavan
