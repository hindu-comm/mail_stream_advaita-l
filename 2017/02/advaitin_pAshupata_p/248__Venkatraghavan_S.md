SaguNa upAsana can only take one up to brahmaloka via shukla gati. An
ajnAni in manushya loka will remain an ajnAni in kailASa also. So even with
krama mukti, jnAna will have to be generated by vedAnta shravaNa, manana,
nidhidhyAsana in brahmaloka.
