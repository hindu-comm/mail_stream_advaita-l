> Here is a passage which contains a definition of atmanubhava:
>
> BSB 4.1.2:
>
> तस्मात् *सर्वदुःखविनिर्मुक्तैकचैतन्यात्मकोऽहमित्येष आत्मानुभवः *।

​This is a response to the objection that such a दुःखित्वाद्यभावं ​न
प्रतिपद्यते, so it is not a definition of अनुभव, but shows that एष अनुभवः
this anubhava is possible. As you say with the later quote, this too is
अनुभवप्रकार।
